# 🌱 Huertas TIC LMS

> **Cultivando conocimiento mediante la tecnología.**

Huertas TIC LMS es una plataforma web de gestión del aprendizaje (Learning Management System - LMS) diseñada para apoyar procesos de formación relacionados con **huertas urbanas, agricultura sostenible y tecnologías aplicadas al sector agrícola**.

La plataforma busca facilitar el acceso al conocimiento mediante contenidos interactivos, actividades de aprendizaje, foros colaborativos y un proyecto final integrador, promoviendo el aprendizaje autónomo y el trabajo colaborativo.

---

## 📖 Descripción

El proyecto fue desarrollado como una plataforma educativa ligera basada en tecnologías web, permitiendo la gestión de cursos, usuarios y actividades sin necesidad de una base de datos o un servidor complejo.

Está orientado a instituciones educativas, docentes y estudiantes interesados en fortalecer competencias relacionadas con:

- 🌱 Agricultura urbana
- 🌾 Agricultura sostenible
- 💧 Uso eficiente del agua
- 🤖 Agricultura inteligente
- 📚 Aprendizaje virtual
- 🌍 Educación ambiental

---

## 🌿 Proyecto Final

El examen tradicional fue reemplazado por un **Proyecto Final Integrador**, donde el estudiante entrega un único enlace que contiene todas las evidencias desarrolladas durante el curso.

El enlace puede incluir:

- Informe final.
- Evidencias fotográficas.
- Video del proyecto.
- Código fuente (si aplica).
- Presentaciones.
- Documentación adicional.

---

- Interactuar con sus compañeros.
- Fortalecer el aprendizaje colaborativo.

# 🚀 Instalación

1. Clonar el repositorio

```bash
git clone https://github.com/juanjmartinezr-source/huertas-tic-lms.git
```

2. Ingresar al directorio

```bash
cd huertas-tic-lms
```

3. Abrir el proyecto con Visual Studio Code.

4. Ejecutar utilizando Live Server o cualquier servidor web local.
